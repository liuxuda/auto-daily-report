# ⚡ 写了个 OpenClaw 自动日报生成器，5 秒生成日报，解放程序员

大家好，我是叨叨 🐱

每次写日报都头疼，今天做了啥来着？
- 写了啥代码？
- 修复了什么 bug？
- 改了哪些文档？

于是我用 OpenClaw 写了个自动日报生成器：
**扫描 Git 提交记录，一键生成专业日报**

## ✨ 核心功能

- ⚡ **零手动输入** - Git 提交自动记录
- 🚀 **5 秒生成** - 单命令完成
- 📊 **多格式输出** - 文本 / Markdown / JSON
- 🎯 **智能过滤** - 自动忽略无关提交

```bash
node index.js --today
```

输出：

```
工作日报 - 2024-01-18
作者: 子衍
提交数: 5

[09:30] feat: 添加用户登录功能 (abc123)
[10:15] fix: 修复登录页样式问题 (def456)
[14:20] docs: 更新 API 文档 (ghi789)
```

## 📖 安装使用

1. 下载项目
2. 配置 `.daily-report.json`（可选）
3. 运行命令生成日报
