## 🎯 产品介绍

每次写日报都头疼？"今天做了啥"想破头？
这个工具帮你自动生成日报，零手动输入，5秒搞定！

---

## ✨ 核心功能

✅ **自动收集** - 扫描Git提交记录，无需手工回忆
✅ **智能过滤** - 自动忽略 test/node_modules 等无关提交
✅ **多格式输出** - 文本/Markdown/JSON 三种格式
✅ **一键生成** - 单命令完成，5秒搞定

---

## 🚀 使用演示

运行命令：
```bash
node index.js --today
```

输出结果：
```
工作日报 - 2024-01-18

提交数: 5

[09:30] feat: 添加用户登录功能

[14:20] docs: 更新API文档
[16:40] refactor: 优化代码结构
[17:20] test: 添加单元测试
```
